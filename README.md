# Sagara Takumi Portfolio

個人のポートフォリオサイト「Sagara Takumi Portfolio」のソースコードです。


## 技術スタック

| レイヤー | 使用技術 | デプロイ先 |
|---------|---------|------------|
| フロントエンド | Next.js (App Router, Static Export) | Cloudflare Pages |
| バックエンド | Hono (TypeScript) | Cloudflare Workers |
| データベース | Cloudflare D1 (SQLite) | Cloudflare D1 |
